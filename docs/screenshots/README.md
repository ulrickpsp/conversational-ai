## Tips for Best Screenshots

1. **Use a clean browser window** ‚Äî No bookmarks bar, extensions, or distracting tabs
2. **Zoom level:** 100% (or 125% for high-DPI displays)
3. **Theme:** The app uses dark mode, make sure it's clearly visible
4. **Sample content:** Use an interesting proposal (e.g., "Create a triangular arbitrage strategy between DEX and CEX for stablecoins") to show meaningful debate content
5. **File format:** PNG for best quality
6. **Compression:** Optimize PNGs with tools like TinyPNG before committing to keep repo size reasonable

## Tools for Capturing Screenshots

### macOS
- **Cmd + Shift + 4**: Select area to capture
- **Cmd + Shift + 5**: Screenshot toolbar with options

### Windows
- **Windows + Shift + S**: Snipping tool
- **Win + PrtScn**: Full screen capture

### Linux
- **Flameshot**: Powerful screenshot tool
- **GNOME Screenshot**: Built-in tool

### Browser Extensions
- **Awesome Screenshot** (Chrome/Firefox)
- **Fireshot** (Chrome/Firefox)
- **GoFullPage** (Chrome) ‚Äî for capturing full-page scrolling screenshots
